# Building Web APIs with Node.js

Шаблонний репозиторій для викладача дисципліни **Building Web APIs with Node.js** (BetterED Neoversity).

## Як почати

Натисни **Use this template** на GitHub щоб створити власний репозиторій з цього шаблону.

Перед кожним заняттям відкрий план у папці `lessons/` та створи нову гілку:

```bash
git checkout -b lesson-01
```

Після заняття закомть та запуш зміни:

```bash
git add .
git commit -m "lesson-01: вступ до Node.js"
git push -u origin lesson-01
```

Наступне заняття починай з поточної гілки:

```bash
git checkout -b lesson-02
```
